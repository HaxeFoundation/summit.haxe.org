Simon Krajewski

Maintainer of the Haxe Compiler

Simon has been the main maintainer of the Haxe Compiler since 2012 and is a founding member of the Haxe Foundation. If he's not fixing Haxe issues, he's probably thinking about how to fix Haxe issues. He also enjoys language and API designing, especially when others do it.
